# University Management System (Java)

A console-based Java application for managing students, teachers, courses, enrollments, and grades.  
Built to demonstrate OOP fundamentals, collections, exceptions, file I/O, and clean architecture.

## Features
- Abstract class `Person` with `Student` and `Teacher` subclasses
- Encapsulation + polymorphism (`toString` overrides)
- Collections: `ArrayList`, `HashMap`, `Set`
- Custom exceptions and validation
- File persistence (save/load)
- Menu-driven console UI
- Enums for `Grade` and `CourseType`
- `LocalDate` for enrollment dates

## Project Structure
```
src/
  university/
    Main.java
    contract/
    exception/
    model/
    service/
    util/
```

## How to Run
1. Open the project in your IDE (IntelliJ / Eclipse / VS Code)
2. Run `Main.java`

## Example Usage
- Add students and teachers
- Add courses
- Enroll students into courses
- Assign grades
- Save & load data automatically

## Technologies
- Java (core)
- OOP, Collections, Exception Handling, File I/O
